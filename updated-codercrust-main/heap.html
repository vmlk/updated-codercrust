Heap Data Structure -

Last Updated: 09july 2021


A Heap is a special Tree-based data structure in which the tree is a complete binary tree.
Generally, Heaps can be of two types:
  Max-Heap: In a Max-Heap the key present at the root node must be greatest among the keys present at all of itâ€™s children.
  Min-Heap: In a Min-Heap the key present at the root node must be smallest among the keys present at all of it's children.

A Heap data structure is a binary tree with the following properties:
  1. It is a complete binary tree: that is, each level of the tree is completely filled, except possibly the bottom level. At this level, it is filled from left to right.
  2. Is satisfies the heap-order property: The data item stored in each node is greater that or equal lto the data items stored in its children.
