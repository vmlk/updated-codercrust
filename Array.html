Array Data Structure -

Last Updated: 13june 2021


An Array is a collection of variable of same data types. All the elements of array are stored in the contiguous memory locations. The size of the array must be a constant integeral value. Individual elements in an array can be accessed by the name of the array and an integer enclosed in square bracket called subscript/index variable like arr[4].





Element − Each item stored in an array is called an element.
Index − Each location of an element in an array has a numerical index, which is used to identify the element.

Basic Operations

Following are the basic operations supported by an array.
    1. Traverse − print all the array elements one by one.
    2. Insertion − Adds an element at the given index.
    3. Deletion − Deletes an element at the given index.
    4. Search − Searches an element using the given index or by the value.
    5. Update − Updates an element at the given index.

2. Insertion operation -

Insert operation is used to insert one or more data elements into an array.Based on the requirement, a new element can be added at the beginning, end, or any given index of array.

Insert Element in an Array-
  a. Insert element at the beginning of an array

  b. Insert element at the end of an array

  c. Insert element at any particular position of an array


2. Deletion operation -

Deletion operation is used to delete data elements into an array.Based on the requirement, deletion can be at the beginning, end, or any given index of array.

Delete Element in an Array-
  a. Delete element from any position of an array
