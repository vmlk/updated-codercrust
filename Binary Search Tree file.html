Data Structure Binary Search Tree(BST) -

Last Updated: 14june 2021


Binary Search Tree(BST) is an non-linear data type. Binary Search tree is a node-based binary tree data structure It divides its subtrees into two segments left subtree right subtree as per the following properties :
      1. The value of the key of the left node data is less then the value of its parent node data.
      2. The value of the right node data is greater then the value of its parent node data.
