Stack Data Structure

Last Updated: 14june 2021


A Stack is a linear data structure which follows the order of LIFO (last in first out) or FILO (first in last out) for performing any particular order. In real Life we can consider the example of plates - the plate which is kept last is removed first and the plate kept first is at the bottom hence removed last, which is a clear explaination of order LIFO/FILO.


Basic operations performed on stack :

1-PUSH()  : Adds an item into the stack . Overflow condition occur if stack is full.
2-POP()  : Removes an item from the stack . Underflow condition occur if stack is empty.
3-TOP()/PEEK()  : Returns the top element from stack.
4-ISEMPTY()   : Returns True if stack id full else false.
